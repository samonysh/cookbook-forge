# 附录 A · functools 常用装饰器速查

Python 标准库 `functools` 提供了几个常用装饰器/工具，自己写装饰器时几乎都会用到。

## 速查表

| 装饰器 / 函数 | 用途 | 典型写法 |
|---|---|---|
| `@functools.wraps(func)` | 拷贝原函数 `__name__`/`__doc__` | wrapper 定义处加上 |
| `@functools.lru_cache(maxsize=N)` | 记忆化（LRU 缓存） | 递归、纯函数 |
| `@functools.cache` | Python 3.9+，等价无上限 LRU | 递归、纯函数 |
| `@functools.singledispatch` | 按第一个参数类型分派（泛函数） | 多态工具函数 |
| `@functools.total_ordering` | 补全比较运算符 | 只写 `__eq__`+`__lt__` 即可 |
| `functools.partial(fn, *args)` | 偏函数：冻结部分参数 | 回调、工厂 |
| `functools.reduce(fn, iter)` | 累积归约 | 替代简单 for-累加 |

## 使用示例

```python
# file: examples/functools_demo.py
import functools

@functools.lru_cache(maxsize=128)
def fib(n: int) -> int:
    """记忆化斐波那契，复杂度从 O(2^n) 降到 O(n)。"""
    if n < 2:
        return n
    return fib(n - 1) + fib(n - 2)

print(fib(40))  # 瞬间出结果；没有 lru_cache 要算很久

@functools.total_ordering
class Version:
    def __init__(self, major: int, minor: int):
        self.major, self.minor = major, minor
    def __eq__(self, other): return (self.major, self.minor) == (other.major, other.minor)
    def __lt__(self, other): return (self.major, self.minor) < (other.major, other.minor)
    # 有了 total_ordering，>=、<=、> 自动生成

print(Version(1, 2) <= Version(1, 10))  # True
```
